# 💰 CLI Expense Tracker

A Python command-line tool to track daily expenses with 
category-wise analysis and persistent CSV storage.

## Features
- Add expenses with category, amount and description
- View all expenses in a formatted table
- Category-wise spending summary
- Delete expenses by ID
- Data persists permanently in CSV

## Tech Stack
- Python 3.x
- Pandas — data storage and analysis

## How to Run
```bash
pip install pandas
python main.py
```
